- 분대 명령
  - 집결
  - 고정
  - 돌격
- 적 경비 AI
- 적 병종 다양화
  - 일반 소총수

  - 유탄병
  - 지정사수

## 조작

### PC
- `WASD`: 이동
- `마우스`: 조준
- `좌클릭`: 사격
- `E`: 상호작용
- `Space`: 스킬
- `R`: 재장전
- `1/2/3`: 분대 명령 전환

### 모바일
- 좌측 가상 스틱: 이동
- `사격`: 발사
- `스킬`: 병과 스킬 사용
- `작전`: 자료 회수 / 탈출

## 실행 방법

가장 간단한 방법:

1. `index.html` 파일을 브라우저로 열기

또는 로컬 서버 사용:

```powershell
cd C:\Users\User\.cokacdir\workspace\jykurxqg
python -m http.server 8080
```

그 후 브라우저에서:

`http://localhost:8080`

## 1차 플레이테스트 체크포인트

처음 플레이하는 유저에게 아래를 확인받으면 좋습니다.

1. 병과 차이가 체감되는지
2. 적 병종 차이가 보이는지
3. 조작이 직관적인지
4. 정보 탈취 / 방어 미션이 서로 다르게 느껴지는지
5. 성공/실패 이유가 명확한지
6. 결과 등급과 통계가 납득되는지
