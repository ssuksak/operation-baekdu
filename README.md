## 조작

### PC
- `WASD`: 이동
- `마우스`: 조준
- `좌클릭`: 사격
- `E`: 상호작용
- `Space`: 스킬
- `R`: 재장전
- `1/2/3`: 분대 명령 전환
- `P` / `Esc`: 일시정지
- `G`: 목표 화살표 토글
- `H`: HUD 토글
- `M`: 미니맵 토글
- `N`: 난이도 변경
- `O`: 오디오 토글
- `F`: 전체화면
- `J`: 성능 통계 토글

### 모바일
- 좌측 가상 스틱: 이동
- 우측 화면 드래그: 조준
- `사격`: 발사
- `스킬`: 병과 스킬 사용
- `작전`: 자료 회수 / 탈출

## 현재 플레이 감각 핵심

- 제한된 시야와 넓은 전장에서 탐색 / 교전 / 탈출 흐름
- 병과별 역할 차이와 분대 명령 활용
- 적 병종 차이, 엄폐, 측면 기동, 수색 압박
- 미션별 핵심 목표 + 보너스 목표 동시 추적

## 1차 플레이테스트 체크포인트

처음 플레이하는 유저에게 아래를 확인받으면 좋습니다.

1. 병과 차이가 체감되는지
2. 적 병종 차이가 보이는지
3. 조작이 직관적인지
4. 정보 탈취 / 정찰 소탕 / 방어 미션이 서로 다르게 느껴지는지
5. 성공/실패 이유가 명확한지
6. 결과 등급과 통계가 납득되는지
7. 첫 판 튜토리얼이 도움 되는지
8. 성능이 불안할 때 경량 모드가 체감되는지

## 한 판 길이 목표

- 정보 탈취: 탐색 + 교란기 + 자료 회수 + 탈출 구조로 5분 이상 플레이 목표
- 정찰 소탕: 정찰 지점 A/B 확보 + 탈출 구조로 5분 내외 전술 플레이 목표
- 전초기지 방어: 장시간 웨이브 방어로 5분 이상 플레이 목표
